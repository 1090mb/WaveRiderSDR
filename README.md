## Usage

Simply open `index.html` in a modern web browser to start using WaveRiderSDR.

### Frequency Band Selection

1. Use the **Band Selector** dropdown to choose between different frequency bands:
   - **HF (3-30 MHz)**: Shortwave radio, amateur radio, international broadcasting
   - **VHF (30-300 MHz)**: FM radio, TV broadcasting, amateur radio, aviation
   - **UHF (300-3000 MHz)**: TV broadcasting, mobile phones, GPS, WiFi, Bluetooth
   - **SHF (3-30 GHz)**: Satellite communications, radar, 5G, WiFi 6

2. Use the **Frequency Controls** to tune to specific frequencies:
   - Enter a frequency directly in the input field
   - Use the tuning buttons for quick frequency adjustments (+/- 10 MHz, 1 MHz, 100 kHz)

### Band Plan Display

The band plan visualization shows:
- **Color-coded allocations** for different services (Amateur, Broadcasting, Mobile, etc.)
- **Current frequency marker** (red line) showing your tuned frequency
- **Interactive controls**: Zoom in/out and reset view
- **Frequency scale** with labeled markers
- **Legend** explaining the color coding

**Interaction:**
- **Hover** over any allocation to see its name
- **Click** on any allocation to view detailed information including:
  - Allocation name
  - Service type
  - Frequency range
  - Bandwidth
